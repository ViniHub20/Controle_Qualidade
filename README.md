# Controle_Qualidade

## Descrição

Sistema simples de gerenciamento de tarefas desenvolvido em Python para a disciplina de Controle de Qualidade de Software.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Editar tarefas
- Remover tarefas

## Tecnologias utilizadas

- Python 3
- Git
- GitHub
- Pytest
- GitHub Actions

## Estrutura do projeto

```
Controle_Qualidade/
│
├── src/
│   ├── app.py
│   └── tarefas.py
│
├── tests/
│   └── test_app.py
│
├── .github/workflows/
│   └── python.yml
│
├── requirements.txt
└── README.md
```

## Como executar

```bash
python src/app.py
```

## Como executar os testes

```bash
python -m pytest -v
```
